# Upload Instructions

Upload the contents of this folder to the root of:

`chamaurise/circleoftrust`

For the cleanest Vercel deploy, the GitHub repo root should contain:

- `index.html`
- `styles.css`
- `app.js`
- `assets/`
- `vercel.json`
- `.vercelignore`
- `.gitignore`
- `README.md`

Do not add:

- `server.mjs`
- `package.json` with a `start` script
- `node_modules/`
- preview screenshots
- server logs

After uploading, import the repo in Vercel and deploy it as a static site.
